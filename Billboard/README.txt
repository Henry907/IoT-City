--- how to run billboard with flask server ---

start flask server in water tower pi by typing
python3 app.py
while in the /home/lbc2/Desktop/webapp/ directory

once server is up, connect mouse and keyboard to billboard pi
open visual studio code (vscode)
open the BillboardManager folder in vscode if not already open
click on "display.html" in vscode
at the bottom right, click on the live server button
it should open a chrome tab, fullscreen it with the f11 key
